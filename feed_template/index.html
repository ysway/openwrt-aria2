openwrt-aria2 Package Feed

This feed provides aria2-static packages for OpenWrt.

Usage

This site root is a landing page. The actual opkg feed is platform-specific.

# Example: x86_64
src/gz aria2-static https://ysway.github.io/openwrt-aria2/x86_64

# Example: aarch64_cortex-a53
src/gz aria2-static https://ysway.github.io/openwrt-aria2/aarch64_cortex-a53

# Then install:
opkg update
opkg install aria2-static

Available Packages

Browse the platform directories in the feed branch for available architectures and their Packages.gz indexes.
