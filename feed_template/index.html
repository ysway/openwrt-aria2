openwrt-aria2 Package Feed

This feed provides aria2-static packages for OpenWrt.

Usage

Add this feed to your OpenWrt device:

# Add to /etc/opkg/customfeeds.conf:
src/gz aria2-static https://ysway.github.io/openwrt-aria2

# Then install:
opkg update
opkg install aria2-static

See the Packages index for available architectures.
